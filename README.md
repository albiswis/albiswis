# AlbisWis (Next.js)

## Run

```bash
npm install
npm run dev
# open http://localhost:3000
```

## Pages

- `/` — Home (hero video, product carousel, "Our Software", partner quotes, CTA)
- `/triton` — Healthcare AI operating system
- `/warpspeed` — Forge: Manufacturing AI platform
- `/atlas` — Real Estate property operations & portfolio intelligence
- `/voya` — Logistics & supply chain intelligence
- `/platforms/aip`, `/platforms/foundry`, `/platforms/gotham`, `/platforms/apollo` — Platform pages
- `/newsroom` — Press releases
- `/q1-2026-letter` — Q1 2026 Letter to Shareholders
- `/docs/foundry` — Foundry docs index
- `/*` (404) — animated Page Not Found

## Stack

Next.js 15 (App Router) · React 19 · Tailwind 3 · TypeScript.
